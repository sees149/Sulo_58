# Voxel Arena (Bloxd.io esintili)

Tarayıcıda çalışan, Bloxd.io tarzında mini bir voxel prototipi.

## Özellikler
- WASD ile hareket, Space ile zıplama, Shift ile koşu
- Pointer lock ile FPS kamera
- Sol tık ile blok kırma
- Sağ tık ile blok yerleştirme
- Basit yerçekimi + çarpışma

## Çalıştırma
```bash
python3 -m http.server 8000
```

Sonra tarayıcıdan `http://localhost:8000` açın.
